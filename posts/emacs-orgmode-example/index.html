列表

  • 无序列表以 - 或 + 开头
  • 有序列表以 1. 或者 1) 开头
  • 描述列表用 ::
  • 列表层级用缩进体现

—

  • 无序列表 1
    • 无序列表 1.1
    • 无序列表 1.2
  • 无序列表 2
    • 无序列表 2.1
    • …

—

  1. 有序列表 1
    1. 有序列表 1.1
    2. 有序列表 1.2
  2. 有序列表 2
    1. 有序列表 2.1
    2. 有序列表 2.2
  3. 指定序号其实是无效的, 导出 html 时会自动编号 20
  4. 如果确实想指定序号, 使用 [@20] 20
